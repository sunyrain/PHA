# 蛋白-水凝胶吸附项目专家审阅稿：中文关键词与提取字段

项目简称：蛋白水凝胶吸附；英文项目代号：PHA。研究目标：系统调研不同水凝胶对不同蛋白的吸附、弱吸附、选择性吸附和抗吸附作用，同时抽取材料制备、物性参数、蛋白性质、实验条件、吸附结果和机理证据，用于数据驱动设计“对给定蛋白吸附或不吸附”的水凝胶体系。

## 四、纳入和排除规则

| 类型 | 规则 |
| --- | --- |
| 纳入 | 水凝胶与一种或多种蛋白之间存在吸附、结合、摄取、捕获、分离、纯化、识别、污染、低吸附或抗污染实验。 |
| 纳入 | 有定量或半定量结果，例如吸附容量、去除率、表面覆盖量、荧光强度、回收率、纯度、抗污染降低比例或对照比较。 |
| 纳入 | 报告水凝胶物性、蛋白性质、酸碱度、盐浓度、温度、接触时间、流速等可用于机理分析或建模的条件。 |
| 排除或降权 | 只有蛋白药物释放，没有吸附或结合数据。 |
| 排除或降权 | 只有组织工程或细胞培养，没有蛋白吸附测量。 |
| 排除或降权 | 只有材料合成表征、只有模拟计算、综述、观点、社论、专利或文献计量分析。 |

## 五、数据记录粒度

| 原则 | 说明 |
| --- | --- |
| 单条记录定义 | 一条记录 = 论文 + 水凝胶材料 + 蛋白靶标 + 实验条件 + 测量结果。 |
| 需要拆分记录的情况 | 同一篇论文比较不同水凝胶、不同蛋白、不同酸碱度、盐浓度、温度、接触时间、浓度、循环次数、流速或洗脱条件。 |
| 不在抽取阶段强行设阈值 | 先保留原始测量值和局部对照，后续根据具体建模任务派生吸附/不吸附标签。 |

## 七、面向反向设计的补充

| 项目 | 需要修改的方向 |
| --- | --- |
| 设计空间表述 | 不再把设计空间理解为“水凝胶材料清单”，而应拆成材料骨架 × 官能化 × 网络结构 × 形态 × 制备工艺 × 使用条件 × 蛋白集合。 |
| 多蛋白目标 | 增加“需要吸附的蛋白集合”和“需要排斥的蛋白集合”，后续模型应学习给定条件下的蛋白响应向量。 |
| 工艺匹配 | 将制备方法、交联/接枝/涂覆条件、洗脱、再生、灭菌、保存和放大风险提升为设计相关字段，而不是普通备注。 |
| 可调变量 | 标记论文中主动改变过的变量，例如酸碱度、盐浓度、温度、配体密度、单体比例、交联密度、孔径或水凝胶形态。 |
| 负样本 | 保留低吸附和抗污染水凝胶，因为它们是“不吸附指定蛋白”的关键训练样本。 |

## 八、希望专家重点审阅的问题

| 序号 | 问题 |
